# Vulnerability Findings

This directory contains all security vulnerability reports found during our DVWA source code review.

## How to Report a Finding

1. Copy the template: `cp template.md sql_injection_login.md`
2. Name files: `[vulnerability_type]_[location].md`
3. Fill out all sections completely
4. Reference exact file paths and line numbers

## File Naming Examples
- `sql_injection_login.md`
- `xss_search.md`
- `csrf_changepw.md`
- `file_upload_gallery.md`
